# 数据源规划(基于《花语簿·数据源调研报告》2026-07)

当前语料为按调研报告 M1 路线**手工整理的演示数据**(25 种花材 + 4 篇花艺理论),尚未接入外部数据源。本文档记录调研结论中与本项目相关的合规路径,供后续扩展参考。

## 调研报告的核心结论

1. **合规是首要筛选维度**:日本花言葉热站(hananokotoba.com 等)明确禁止转载/商用;iPlant.cn《中国植物志》版权归中科院植物所,商用需机构合作。花语字段必须自建或走付费授权(TianAPI),不能爬取充数。
2. **公有领域是花语的"合规金矿"**:Latour《Le langage des fleurs》(1819)与 Tyas《The Language of Flowers》(1840s)已入公有领域,BHL 有扫描件,可自由数字化、翻译、商用。本项目西方花语溯源即引用此传统。
3. **理论层无版权障碍**:AIFD 十项评估原则、60-30-10 配色法则、池坊/小原流/草月流的历史与主张均为事实性内容,可自由用作方案骨架(不复制教材原文)。本项目 `data/corpus/knowledge/` 即按此落地。
4. **行情层有政府一手源**:昆明市农业农村局每日发布斗南花市价格(结构化 HTML 表格,.gov.cn 公开数据,署名即可用)——若做"花价"功能,这是唯一同时满足一手/日更/结构化/合规的源。

## 对本项目的分阶段接入建议(与调研报告 M1-M3 对齐)

- **已做(对应 M1)**:手工整理常见花材(中文名/学名/科属/花期/颜色/场景),静态录入 AIFD + 60-30-10 + 三大流派,西方花语标注 Latour/Tyas 溯源。
- **候选(对应 M2)**:GBIF API 按 `license IN (CC0, CC_BY)` 过滤做学名校验;Wikidata SPARQL 拉通 POWO/GBIF/iNaturalist ID;iNaturalist v2 API 补 CC0/CC BY 图片(注意图片版权独立于数据)。
- **远期(对应 M3)**:昆明斗南日报爬虫(≤10 req/min,UA 带联系邮箱)做行情时序表;中科院植物所机构合作。

## 合规红线(引用自报告 Checklist)

- 日本花言葉站点内容一律不抓取,花语自建
- PPBC/iNaturalist 图片版权归摄影师,商用逐张确认许可
- 电商平台(花点时间/花加)SKU 与价格不抓取
- 政府 .gov.cn 数据署名"数据来源:昆明市农业农村局"即可使用
